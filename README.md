# 🥄 내 수업에 코딩 한 스푼

> **AI디지털수업평가전문가연수 · 러버블 바이브코딩 강의용 웹앱**  
> 교사가 개발자가 되는 것이 아니라, AI와 함께 수업의 문제를 정의하고 실제로 작동하는 수업 도구를 만들어 보는 6시간 실습형 연수 포털입니다.

---

## 1. 프로젝트 소개

**내 수업에 코딩 한 스푼**은 바이브코딩을 처음 접하는 교사들이  
수업 아이디어를 웹앱으로 구현하는 전체 흐름을 경험할 수 있도록 만든 연수용 웹앱입니다.

이 웹앱은 단순히 코딩 도구 사용법을 안내하는 자료가 아니라,  
교사가 자신의 수업 문제를 발견하고, PRD를 작성하고, 생성형 AI 도구를 활용해  
작동하는 수업 도구를 완성하도록 돕는 실습형 포털입니다.

핵심 메시지는 다음과 같습니다.

> 좋은 앱보다 좋은 수업 문제 정의가 먼저입니다.  
> 교사는 개발자가 아니라 교육 문제 해결자입니다.

## 3. 주요 기능

### ✅ 6시간 연수 흐름 제공

연수 전체를 4개의 Phase로 나누어  
교사가 단계적으로 바이브코딩을 이해하고 실습할 수 있도록 구성했습니다.

| Phase | 시간 | 주제 |
|---|---:|---|
| Phase 1 | 60분 | 관점 바꾸기 |
| Phase 2 | 120분 | 첫 번째 완성 경험 |
| Phase 3 | 60분 | 설계하고 확장하기 |
| Phase 4 | 120분 | 내 수업 도구 완성하기 |

---

### ✅ 모듈별 학습 구조

각 Phase 안에는 연수 흐름에 맞는 모듈이 배치되어 있습니다.

- 바이브코딩의 의미 이해
- 교사에게 필요한 바이브코딩 관점 정리
- 프론트엔드, 백엔드, 데이터베이스, API 등 핵심 용어 안내
- Canva Code, Claude Artifacts, Google AI Studio, Lovable, Claude Code 등 도구 소개
- Lovable을 활용한 첫 번째 웹앱 제작
- PRD 작성과 기능 설계
- 백엔드와 생성형 API의 역할 이해
- 수업 도구 프로토타입 제작
- 개인정보 보호 및 디지털 윤리 점검

---

### ✅ 교사용 자료실 제공

연수 중 바로 활용할 수 있는 자료를 웹앱 안에서 제공합니다.

- 실습용 프롬프트
- PRD 작성 가이드
- 기능 설계 체크리스트
- 디지털 윤리 점검표
- Markdown 형식 다운로드 자료

---

### ✅ 학습 진행 관리

연수 참여자가 자신의 진행 상황을 확인할 수 있도록  
모듈 완료 표시와 전체 진행률 확인 기능을 포함했습니다.

- 모듈별 완료 체크
- 완료 상태 저장
- 연수 흐름에 따른 단계적 진행
- 처음 사용하는 교사도 따라가기 쉬운 구조

## 6. 개발 과정

이 프로젝트는 다음 흐름으로 제작되었습니다.

1. **Lovable에서 초기 웹앱 구조 제작**
2. 연수 흐름에 맞게 Phase와 모듈 구성
3. 교사용 자료실, 프롬프트, 체크리스트 기능 추가
4. 개인정보처리방침과 사용약관 페이지 구성
5. Claude Code로 프로젝트를 옮겨 세부 구조와 문구 정리
6. GitHub 레포지토리에 업로드
7. Cloudflare Pages를 통해 최종 배포

## 7. 사용 방법

1. 배포 주소에 접속합니다.  
   https://enlovable.labbitory.com/

2. 연수 흐름에 따라 Phase 1부터 순서대로 진행합니다.

3. 각 모듈의 안내를 읽고 실습 활동을 수행합니다.

4. 필요한 경우 자료실에서 프롬프트와 체크리스트를 다운로드합니다.

5. 모듈을 완료하면 완료 표시를 눌러 진행 상황을 확인합니다.

6. 마지막 단계에서 자신의 수업 도구 아이디어를 정리하고 프로토타입 제작까지 진행합니다.

## 8. 프로젝트 의의

이 웹앱은 교사들이 코딩 자체를 배우는 것을 목표로 하지 않습니다.

대신 다음 경험을 제공하는 데 목적이 있습니다.

- 수업 문제를 스스로 정의하기
- 생성형 AI와 협업하여 아이디어를 구체화하기
- 작은 웹앱을 직접 만들어 보기
- 개인정보 보호와 디지털 윤리를 함께 점검하기
- 실제 수업에 적용 가능한 도구로 발전시키기

바이브코딩은 교사가 개발자가 되기 위한 과정이 아니라,  
교사가 자신의 수업을 더 잘 설계하기 위한 새로운 도구입니다.
